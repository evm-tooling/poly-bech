## Feature Reference

The table below lists the main DSL features and links to their full guides. Each feature has its own page with syntax, examples, and gotchas.

| Feature | Description | Full Guide |
|---------|-------------|------------|
| **Suites** | Group benchmarks with shared config; declare suiteType, runMode, sameDataset | [Suites](/docs/features/suites) |
| **Setup Blocks** | Per-language imports, declarations, init, and helpers | [Setup Blocks](/docs/features/setup) |
| **Benchmarks** | Language implementations, lifecycle hooks, overrides, benchAsync | [Benchmarks](/docs/features/benchmarks) |
| **Fixtures** | Shared test data — inline hex, @file, data/encoding, structured extraction | [Fixtures](/docs/features/fixtures) |
| **globalSetup** | One-time setup before benchmarks; Anvil spawning | [globalSetup](/docs/features/global-setup) |
